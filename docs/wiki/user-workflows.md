# 사용자 흐름

## 운영자 쿠폰 발급

운영자 또는 슈퍼 관리자는 대상 사용자, 쿠폰명, 정액·정률 할인값, 최소 주문금액, 최대 할인액, 유효기간을 지정해 쿠폰을 발급한다. 권한 없는 요청은 거부된다.

## 주문 쿠폰 적용

사용자가 주문에 쿠폰을 연결하면 결제 직전에 쿠폰을 임시 적용한다. 소유자, 유효기간, 사용 상태, 최소 주문금액 조건을 등록 순서대로 평가한다. 모두 통과하면 할인액을 주문 금액에 반영하고 적용 기록을 남긴다.

결제 성공은 쿠폰과 적용 기록을 사용 완료로 확정한다. 결제 실패는 쿠폰을 다시 사용 가능 상태로 복구한다.
