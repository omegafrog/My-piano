# UC-002. 사용자가 주문 결제에 할인 쿠폰을 적용한다

## 목표

- 사용자가 보유한 할인 쿠폰을 주문 결제에 적용하여 할인받는다.

## 사전 조건

- 사용자는 할인 쿠폰의 쿠폰 보유자다.
- 할인 쿠폰은 유효기간 내이고 미사용 상태다.
- 주문금액이 할인 쿠폰의 최소 주문금액을 충족한다.

## 기본 흐름

1. 사용자가 주문 결제 직전에 보유한 할인 쿠폰을 선택한다.
2. 시스템이 할인 쿠폰의 유효기간, 사용 상태, 최소 주문금액을 확인한다.
3. 시스템이 정액 할인 또는 정률 할인 방식으로 할인액을 계산한다.
4. 정률 할인 결과가 최대 할인액을 초과하면 시스템이 최대 할인액까지만 할인한다.
5. 시스템이 할인 쿠폰을 주문 결제에 임시 적용하고 할인액을 반영한다.
6. 결제가 성공하면 시스템이 할인 쿠폰을 사용 처리한다.

## 성공 기준

- 조건을 충족한 할인 쿠폰의 할인액이 주문 결제에 반영된다.
- 결제 성공 후 할인 쿠폰은 다시 사용할 수 없다.

## 실패 기준

- 유효기간이 만료됐거나 이미 사용됐거나 최소 주문금액에 미달한 할인 쿠폰은 적용되지 않는다.
- 임시 적용 뒤 결제가 실패하면 시스템이 쿠폰 적용을 복구하여 할인 쿠폰을 다시 사용할 수 있게 한다.
