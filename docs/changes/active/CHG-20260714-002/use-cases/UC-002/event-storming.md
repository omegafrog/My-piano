# UC-002. 사용자가 주문 결제에 할인 쿠폰을 적용한다 이벤트 스토밍

## 시작 유스케이스

- 유스케이스: UC-002. 사용자가 주문 결제에 할인 쿠폰을 적용한다
- 액터: 사용자
- 목표: 보유한 할인 쿠폰을 주문 결제에 적용하여 할인받는다.
- 초기 커맨드: 🟦 할인 쿠폰을 주문 결제에 적용한다

## 흐름

### 기본 흐름

🟦 할인 쿠폰을 주문 결제에 적용한다 → 🟧 할인 쿠폰이 주문 결제에 임시 적용되었다 → 🟪 쿠폰 보유자, 유효기간, 사용 상태, 최소 주문금액을 충족해야 한다

🟦 결제를 완료한다 → 🟧 결제가 성공했다 → 🟪 결제가 성공하면 임시 적용된 할인 쿠폰을 사용 처리한다

🟦 할인 쿠폰을 사용 처리한다 → 🟧 할인 쿠폰이 사용 처리되었다 → 🟪 사용 처리된 할인 쿠폰은 다시 사용할 수 없다

### 예외 흐름

🟦 할인 쿠폰을 주문 결제에 적용한다 → 🟧 할인 쿠폰 적용이 거부되었다 → 🟪 쿠폰 보유자, 유효기간, 사용 상태, 최소 주문금액 중 하나라도 충족하지 않으면 적용하지 않는다

🟦 결제를 완료한다 → 🟧 결제가 실패했다 → 🟪 결제가 실패하면 임시 적용된 할인 쿠폰을 복구한다

🟦 할인 쿠폰 적용을 복구한다 → 🟧 할인 쿠폰 적용이 복구되었다 → 🟪 복구된 할인 쿠폰은 다시 사용할 수 있다

## 불변식

- 할인 쿠폰은 쿠폰 보유자만 주문 결제에 적용할 수 있다.
- 유효기간이 만료됐거나 사용 처리된 할인 쿠폰은 적용되지 않는다.
- 최소 주문금액에 미달한 주문에는 할인 쿠폰이 적용되지 않는다.
- 정률 할인액은 최대 할인액을 초과하지 않는다.
- 결제가 성공한 할인 쿠폰은 다시 사용할 수 없다.
- 결제가 실패한 임시 적용 할인 쿠폰은 다시 사용할 수 있다.
