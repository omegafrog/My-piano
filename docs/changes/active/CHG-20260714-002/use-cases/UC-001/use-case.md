# UC-001. 운영자가 사용자에게 할인 쿠폰을 발급한다

## 목표

- 운영자가 특정 사용자에게 할인 쿠폰을 수동 발급한다.

## 사전 조건

- 운영자는 발급할 대상 사용자를 지정할 수 있다.
- 할인 쿠폰의 할인 방식, 최소 주문금액, 최대 할인액, 유효기간, 1회 사용 조건이 정해져 있다.

## 기본 흐름

1. 운영자가 대상 사용자를 지정한다.
2. 운영자가 할인 쿠폰의 조건을 입력한다.
3. 운영자가 쿠폰 발급을 요청한다.
4. 시스템이 지정된 사용자에게 할인 쿠폰을 발급한다.

## 성공 기준

- 지정된 사용자가 발급된 할인 쿠폰의 쿠폰 보유자가 된다.

## 실패 기준

- 대상 사용자에게 할인 쿠폰이 발급되지 않는다.
