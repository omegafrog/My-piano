# UC-001. 운영자가 사용자에게 할인 쿠폰을 발급한다 이벤트 스토밍

## 시작 유스케이스

- 유스케이스: UC-001. 운영자가 사용자에게 할인 쿠폰을 발급한다
- 액터: 운영자
- 목표: 특정 사용자에게 할인 쿠폰을 수동 발급한다.
- 초기 커맨드: 🟦 할인 쿠폰을 발급한다

## 흐름

### 기본 흐름

🟦 할인 쿠폰을 발급한다 → 🟧 할인 쿠폰이 발급되었다 → 🟪 대상 사용자와 할인 조건이 모두 지정되어야 한다

### 예외 흐름

🟦 할인 쿠폰을 발급한다 → 🟧 할인 쿠폰 발급이 거부되었다 → 🟪 대상 사용자 또는 할인 조건이 지정되지 않으면 발급하지 않는다

## 도메인 요소

| 유형 | 내용 | 트리거 | 결과 | 시스템 |
| --- | --- | --- | --- | --- |
| 🟦 커맨드 | 할인 쿠폰을 발급한다 | 운영자가 대상 사용자와 할인 조건을 입력한다 | 할인 쿠폰이 발급되었다 또는 할인 쿠폰 발급이 거부되었다 | 쿠폰 시스템 |
| 🟧 이벤트 | 할인 쿠폰이 발급되었다 | 대상 사용자와 할인 조건이 모두 지정된 발급 요청 | 지정된 사용자가 쿠폰 보유자가 된다 | 쿠폰 시스템 |
| 🟧 이벤트 | 할인 쿠폰 발급이 거부되었다 | 대상 사용자 또는 할인 조건이 누락된 발급 요청 | 대상 사용자에게 쿠폰을 발급하지 않는다 | 쿠폰 시스템 |
| 🟪 정책 | 대상 사용자와 할인 조건이 모두 지정되어야 한다 | 할인 쿠폰 발급 요청 | 발급 가능 여부를 판단한다 | 쿠폰 시스템 |
| 🟩 외부 시스템 | 없음 | - | - | - |

## 불변식

- 발급된 할인 쿠폰의 보유자는 지정된 대상 사용자다.
- 대상 사용자 또는 할인 조건이 지정되지 않은 할인 쿠폰은 발급되지 않는다.
