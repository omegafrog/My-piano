# Ubiquitous Language

## 용어

| 용어 | 분류 | 정의 | 코드 표기 | 금지/대체 표현 |
|---|---|---|---|---|
| 할인 쿠폰 | 도메인 개념 | 운영자가 특정 사용자에게 발급하며, 주문 결제 금액을 할인하는 1회성 혜택 | `DiscountCoupon` | 쿠폰, 할인권 |
| 쿠폰 발급 | 행위 | 운영자가 대상 사용자에게 할인 쿠폰을 수동으로 부여하는 행위 | `issueCoupon` | 쿠폰 생성, 쿠폰 지급 |
| 쿠폰 보유자 | 역할 | 발급된 할인 쿠폰을 사용하도록 지정된 사용자 | `CouponHolder` | 수령자, 대상 회원 |
| 정액 할인 | 할인 방식 | 주문 금액에서 쿠폰에 정의된 고정 금액을 차감하는 할인 방식 | `FIXED_AMOUNT` | 금액 할인, 고정 할인 |
| 정률 할인 | 할인 방식 | 주문 금액에 쿠폰에 정의된 할인율을 적용해 할인액을 계산하는 할인 방식 | `PERCENTAGE` | 퍼센트 할인, 비율 할인 |
| 최소 주문금액 | 적용 조건 | 쿠폰을 적용하기 위해 주문이 충족해야 하는 최소 금액 | `minimumOrderAmount` | 최소 결제금액 |
| 최대 할인액 | 적용 한도 | 정률 할인으로 계산된 할인액이 초과할 수 없는 상한 금액 | `maximumDiscountAmount` | 할인 상한, 최대 할인 금액 |
| 유효기간 | 적용 조건 | 쿠폰을 사용할 수 있는 시간 범위 | `validityPeriod` | 만료일, 사용 기간 |
| 임시 적용 | 결제 단계 | 결제 직전에 쿠폰을 결제에 예약하고, 결제 결과에 따라 사용 또는 복구하는 처리 | `temporarilyApplyCoupon` | 쿠폰 사용, 쿠폰 차감 |
| 쿠폰 사용 처리 | 결제 성공 결과 | 결제가 성공했을 때 임시 적용된 쿠폰을 재사용할 수 없게 확정하는 처리 | `markCouponUsed` | 쿠폰 소진, 사용 완료 |
| 쿠폰 적용 복구 | 결제 실패 결과 | 결제 실패 시 임시 적용을 해제하여 쿠폰을 다시 적용할 수 있게 하는 처리 | `restoreCouponApplication` | 쿠폰 롤백, 사용 취소 |
